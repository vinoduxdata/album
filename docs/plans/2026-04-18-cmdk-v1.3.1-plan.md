# cmdk palette v1.3.1 — admin queue verbs implementation plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add 8 admin-only commands to the cmdk palette that wrap the admin-panel's queue-management primitives (run 5 specific jobs, pause/resume all queues, clear failed jobs across all queues).

**Architecture:** Thin wrapper handlers in `web/src/lib/managers/command-items.ts`. Single-queue commands call `runQueueCommandLegacy` once. Bulk commands (`Pause all` / `Resume all` / `Clear failed jobs`) fan out across a curated 14-queue list via `Promise.allSettled` so one rejection never drops the rest. Prerequisite refactor extracts the admin-panel's implicit queue list into a shared constants module so palette and panel draw from the same source and TypeScript enforces no drift.

**Tech Stack:** SvelteKit 2, Svelte 5, TypeScript strict, `@immich/sdk` (oazapfts-generated), `@immich/ui`, `svelte-i18n`, Vitest + happy-dom. No server/DB/OpenAPI/Dart/E2E changes.

```ts
async function runQueue(name: QueueName) {
  const $t = get(t);
  const item = asQueueItem($t, { name });
  try {
    await runQueueCommandLegacy({
      name,
      queueCommandDto: { command: QueueCommand.Start, force: false },
    });
    toastManager.primary($t('cmdk_cmd_job_started', { values: { job: item.title } }));
  } catch (error) {
    handleError(error, $t('errors.something_went_wrong'));
  }
}
```

**Why `asQueueItem`**: it maps each `QueueName` to the admin panel's noun-form job title (e.g. `admin.thumbnail_generation_job` = "Thumbnail generation"). The toast then reads "Started: Thumbnail generation" instead of "Started: Run thumbnail generation" (which is what passing the verb-form palette label would produce).

## Task 5: Implement bulk Pause/Resume commands

### Step 5.1: Add the `bulkQueueCommand` helper

In `command-items.ts`, directly under `runQueue` (from Task 3), insert:

```ts
async function bulkQueueCommand(command: QueueCommand.Pause | QueueCommand.Resume) {
  const results = await Promise.allSettled(
    ADMIN_VISIBLE_QUEUES.map((name) => runQueueCommandLegacy({ name, queueCommandDto: { command } })),
  );
  const failed = results.filter((r) => r.status === 'rejected').length;
  const $t = get(t);
  if (failed > 0) {
    toastManager.warning($t('cmdk_cmd_bulk_partial', { values: { failed, total: results.length } }));
    return;
  }
  toastManager.primary($t(command === QueueCommand.Pause ? 'cmdk_cmd_all_paused' : 'cmdk_cmd_all_resumed'));
}
```
